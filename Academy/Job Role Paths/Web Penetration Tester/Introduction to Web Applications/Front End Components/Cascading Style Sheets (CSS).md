# Cascading Style Sheets (CSS)

CSS (Cascading Style Sheets) is the stylesheet language used alongside HTML to format and style HTML elements. Each CSS version introduces new capabilities supported by updated browsers.

## Fundamentals

CSS defines the style of HTML elements and classes, applying consistent formatting across a page. This includes font family, size, background color, text color, alignment, and more.

```css
body {
    background-color: black;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: helvetica;
    font-size: 10px;
}
```

Setting unique IDs or class names on HTML elements allows you to reference them later in CSS or JavaScript.

## Syntax

CSS uses curly brackets `{}` to define styles, with properties and values in the format: `element { property: value; }`

HTML elements support hundreds of properties: height, position, border, margin, padding, color, text-align, font-size, and more. These combine to create visually appealing layouts.

CSS supports advanced animations using `@keyframes`, `animation`, `animation-duration`, `animation-direction`, and more.

## Usage

CSS works alongside JavaScript to dynamically adjust styles, perform calculations, and create animations based on user interactions or cursor position.

When combined with HTML and JavaScript, CSS enables visually stunning, user-friendly web applications. CSS also integrates with XML, SVG, and modern mobile development frameworks for UI design.

## Frameworks

Many developers find manual CSS styling inefficient. CSS frameworks provide pre-built stylesheets and components for faster development:

- **Bootstrap**
- **SASS**
- **Foundation**
- **Bulma**
- **Pure**

These frameworks are optimized for web applications and include common elements needed in modern development.
