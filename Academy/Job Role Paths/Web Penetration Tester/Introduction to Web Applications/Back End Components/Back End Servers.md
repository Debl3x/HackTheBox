# Back End Servers

A back-end server is the hardware and operating system hosting all applications necessary to run a web application. It executes all processes and tasks comprising the entire application, fitting within the Data access layer.

## Software

The back end server contains three core components:

- **Web Server**
- **Database**
- **Development Framework**

Additional software may include hypervisors, containers, and WAFs.

### Common Stacks

Popular back-end stack combinations:

| Stack | Components |
|-------|-----------|
| LAMP | Linux, Apache, MySQL, PHP |
| WAMP | Windows, Apache, MySQL, PHP |
| WINS | Windows, IIS, .NET, SQL Server |
| MAMP | macOS, Apache, MySQL, PHP |
| XAMPP | Cross-Platform, Apache, MySQL, PHP/PERL |

## Hardware

The back end server contains all necessary hardware. Hardware power and performance determine application stability and responsiveness.

Large-scale applications often distribute load across multiple back end servers working collectively. Web applications may also utilize data centers and cloud hosting services providing virtual hosts.
